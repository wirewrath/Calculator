# Simple Web Calculator

A clean, responsive web calculator built from scratch using frontend web technologies. This project was developed as a hands-on exploration of user interface design, DOM manipulation, and JavaScript logic handling.

## 🚀 Features
* **Core Math Operations:** Supports addition, subtraction, multiplication, and division.
* **Instant Clear & Delete:** Includes `AC` (All Clear) to reset the display and a backspace feature to delete the last entered digit.
* **Responsive Design:** A beautifully styled grid layout that scales cleanly across mobile and desktop viewports.
* **Read-Only Display:** Prevents direct keyboard input tampering to ensure a controlled user experience through the UI buttons.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for the calculator wrapper, display, and input keys.
* **CSS3:** Custom grid layouts, button hover transitions, and a modern color scheme.
* **JavaScript (ES6):** Dynamic DOM element updates, string expression parsing, and arithmetic logic.

## 💡 What I Learned
* **DOM Manipulation:** Accessing and modifying HTML element values dynamically in real-time based on user click events.
* **Workspace Synchronization:** Overcoming local environment pathing issues to successfully mirror code changes between a local machine and a remote GitHub repository.
* **State Management:** Handling operator inputs and managing how numerical strings chain together before evaluating an expression.

## 📂 Project Structure
* `index.html` - The structural markup and button layout.
* `style.css` - Visual presentation, layout alignment, and UI styling.
* `script.js` - Click event listeners and calculation math logic.
